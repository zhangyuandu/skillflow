# SkillFlow Logo 概念

## 设计理念

**核心元素：**
1. **水流/波浪** - 代表"流动"、"自然"、"连续"
2. **技能图标** - 代表"技能"、"工具"、"能力"
3. **连接/编排** - 代表"协调"、"编排"、"整合"

## Logo 方案

### 方案1: 简约波浪 (推荐)
```
    ~~~~      🔧
   ~~~~~~    📄
  ~~~~~~~~   📊
   ~~~~~~    🔍
    ~~~~
   
   SKILLFLOW
```
**特点:**
- 简约现代
- 易于识别
- 缩放友好

### 方案2: 技能流向
```
    ┌───┐
    │ A │─┐
    └───┘ │
    ┌───┐ │   ┌─────┐
    │ B │─┼──▶│OUT  │
    └───┘ │   └─────┘
    ┌───┐ │
    │ C │─┘
    └───┘
   
   SKILLFLOW
```
**特点:**
- 表达编排概念
- 适合技术场景

### 方案3: 抽象流动
```
   ≋≋≋≋≋≋≋≋≋
   ≋ S F ≋
   ≋≋≋≋≋≋≋≋≋
   
   SkillFlow
```
**特点:**
- 符号化
- 独特记忆点

## 颜色方案

### 主色: 科技蓝
- `#0066FF` - 主蓝
- `#0052CC` - 深蓝
- `#4D94FF` - 浅蓝

### 辅助色
- `#00C9A7` - 青绿 (代表流动)
- `#6C5CE7` - 紫色 (代表智能)
- `#2D3436` - 深灰 (文字)
- `#FFFFFF` - 白色 (背景)

## 应用场景

### GitHub 仓库
- 大小: 400x400 (Profile 图片)
- 格式: PNG/SVG

### README 标题
- 可以使用文字 Logo 配合波浪装饰
- Markdown 中可用 ASCII 或图片

### 社交媒体
- Twitter: 1200x675
- LinkedIn: 1200x627
- 微信朋友圈: 900x500

### 网站/文档
- 顶部导航 Logo: 高度 40px
- 底部 Logo: 高度 30px
- 水印: 透明度 10%
